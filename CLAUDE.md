# CLAUDE.md

Claude Code는 이 프로젝트에서 `AGENTS.md`와 동일한 공통 규칙을 따른다.

## 작업 순서

1. `AGENTS.md`와 `COLLABORATION.md`를 읽는다.
2. 작업 대상 API는 `docs/API_SPEC.md`와 `docs/API_FIELD_GUIDE.md`에서 확인한다.
3. 엔티티와 데이터 변경은 `docs/ERD.md`에서 확인한다.
4. 관련 코드와 테스트를 읽은 후 최소 범위로 구현한다.
5. API 또는 ERD가 바뀌면 문서와 변경 이력을 같은 작업에서 수정한다.
6. 관련 테스트를 실행하고 결과를 보고한다.

## 금지사항

- 명세에 없는 필드, 상태 코드, 인증 방식을 임의로 추가하지 않는다.
- 1차 스프린트에 인증, 편집 토큰, 일정 `version`을 임의로 도입하지 않는다.
- 비밀값을 코드, 문서, 로그, 테스트에 남기지 않는다.
- 요청 없이 CI와 배포 설정을 변경하지 않는다.

완료 시 변경 내용, 검증 결과, API·ERD 변경 여부, 남은 문제를 명시한다.
